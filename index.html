Analysis

Further analysis can be performed on the loaded layer/s (classification, density mapping, time-based tracking, and data profiling), though certain kinds of analysis are only applicable to specific web service layer types (e.g., WFS). This includes:
1. Classification: The classify WFS feature, as the name implies, works on Web Feature Services, due to their nature in containing attribute values and independent, clickable vector features (as opposed to rasterized images which are fuzzier in nature)
2. Density Mapping: The density mapping feature provides an option to select a WFS layer on which to calculate kernel density estimation (where a smoothly curved surface is fitted over each point or line feature, and surface values are higher near the clustering of features, and lower as distances increase from those clusters)
3. Feature Tracking: The tracking tool again takes a web service layer as input, and performs time series-based tracking along the route or direction the selected feature travels as time passes (as an animation)
4. Vertical Profiling: This tool involves the combination of two types of web service layers: WCS and WFS. The WCS layer provides height/elevation information, and the WFS provides the features which experience the height/elevation changes
